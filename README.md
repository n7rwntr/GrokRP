# Prattle (please read anyway):
my set of instructions for RP (more like "interactive fanfic" because zero mechanics) with Grok. for best results, use them in any established grimdark universe with rich training data (The First Law, ASOIAF, Dune etc.).

1. create a project and upload detailed timeline (a great example would be 'ASOIAF Timeline - Vandal Proof' by Matt Hedges) and maps as project files to ground current world state and reduce hallucinating: Grok knows the whole story and tends to jump the gun, always indicate current timeframe when playing.
2. SuperGrok's context window seems to be 128k so after about 40 or so responses, ask Grok to create events summary & end states, save both to a doc, then start new chat/chapter under the same project.

special notes:
- matter of personal taste, but I place an emphasis on living, breathing world that does *not* revolve around PC (Player Character, as I don't call them MC) unless logically justified. so I imagine the instructions will work poorly with power fantasy/harem and such. or you can customise by deleting entries (1.2) and (3.3).
- best with slow burn, long-form RP over dozens of arcs. (my longest running campaign is 60+ chapters with 40+ chapters of alt-timelines... *send help*)
